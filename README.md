# Python Calculator

A simple command-line calculator built with Python.

## Features
- Addition
- Subtraction
- Multiplication
- Division (with zero division protection)

## How to Run
1. Make sure Python is installed
2. Clone this repository
3. Run the file:
python calculator.py
